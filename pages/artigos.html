Introdução a JavaScript

Introdução

JavaScript(JS) é uma linguagem de alto nível muito usada atualmente. Por ser uma linguagem super flexível - podendo atuar tanto no Back-End como no Front-End -, e também por ser compatível com o HTML e com os navegadores, é chamada também de "A linguagem da web". Nesse artigo, vamos aprender como manipular os seus tipos de dados primitivos (data types!).

String

A string é um tipo de dado primitivo do JavaScript, que se pode definir como todo caracter ou conjunto de caracteres que esteja entre aspas. Formalmente, "2" é na verdade uma string, e não um número. Podemos alterar uma string por funções que a própria linguagem nos dá, chamadas de métodos.

Por exemplo, se temos uma string "bolo", e quisermos juntar essa string com outra "de banana", usamos o metódo cat(), do seguinte modo: cat("bolo","de banana"). O resultado sairá "bolode banana", isso porque o espaço é considerado um caracter, por isso se quisermos colocar um espaço entre "bolo" e "de", a string "bolo" deve ser, na verdade, "bolo ".
